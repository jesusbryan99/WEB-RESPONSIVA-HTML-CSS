Descripcion:

Este proyecto fue iniciado el 2 de Septiembre de 2020 y termino el 17 de Diciembre de 2020.
Empresa orientada en la rama de la construccion fundada en 2018.
Se ralizo un sistema interno de escritorio para cumplir con las nececidades de esta misma
Este proyecto se realizo dentro de las instalaciones utilizando equipo de trabajo personal en el cual se utilizo netbeans como IDE de desarrollo, y se utilizo Java como lenguaje de programacion y MySQL como base de datos este proyecto se subio en un servidor interno de la empresa.

  • Almacenamiento de documentos de facturacion
  • Control de creditos
  • Control de inventario
  • Control de personal
  • Control clientes y proveedores
  • Generador de reportes por obra en PDF y CVS
  • Control de facturas ingresadas y expedidas
  • Incio de sociones personalizadas y configuracion de permisos
  • Incio de sociones personalizadas y configuracion de permisos
  • Reporte general de utilidades netas de la empresa por obra
  • Recordatorios

Sistema para avaluos

proyecto realizado como freelace

Descripcion

Se realizo un sistema basado en avaluos el cual ya se tenia en un formato de Excel el cual se buscaba se convirtiera a un sistema el cual pudiese almacenar los avaluos y abrirlos en una misma pagina y optimizar la manera de realizar estos procesos ademas de optimizar el almacenado y ajuste de imagenes.
Para este proyecto se utilizo bootstrap para el diseño de formularios, JavaScript como lenguaje de programacion y MySQL como base de datos.

    Tareas que se realizaron

  • Adquisicion y analisis de requerimentos
  • Creacion de formularios
  • Creacion de permisos y seguimiento de movimientos
  • Diseños iterativos
  • Pruebas de usabilidad
  • Realizacion de reportes por dia, semana y mes.
  • Se realizo proceso de costos
  • Seguimiento de Avaluos y cotizacion de costos
